# 조합 폭발 테스트

사람이 수동으로 나열하기 어려운 조건 축의 곱집합에서 놓치기 쉬운 테스트 케이스를 만든다.
예를 들어 타겟 없음, 타격수 0, PvP Off, 광역 스킬처럼 각각은 단순하지만 함께 만나면
규칙의 빈틈이나 우선순위 충돌이 드러나는 조합을 찾는다.

## Input

- 조건 축 목록 (타겟 유무 / 타격수 / PvP 상태 / 지역 / 스킬 유형 등)
- 각 축의 값 범위와 금지 조합
- 반드시 지켜야 하는 불변 조건 또는 기대 결과

## Output

- 사람이 놓치기 쉬운 조합 후보
- 각 조합이 위험한 이유와 검증해야 할 기대 결과
- 기존 테스트와 중복되지 않는 신규 케이스 목록

## Zero residue

잔존 0 = **추가 조합 없음**. 정의된 조건 축의 조합이 이미 테스트되었거나 명시적으로 제외되어
새로 생성할 케이스가 없다는 신호.

# 리스크 기반 추천

기능·스킬·아이템·정책 변경이 여러 위험 축에서 충돌할 가능성을 평가하고, 우선적으로 확인할
검증·기획·운영 액션을 추천한다. 단순 영향 목록이 아니라 PvP, 안전지역, 아군 타격처럼 사고가
커질 수 있는 축을 기준으로 우선순위를 매긴다.

## Input

- 평가 대상 (스킬 / 시스템 / 아이템 / 정책 / 콘텐츠)
- 관련 위험 축 (PvP, 안전지역, 아군 타격, 경제, 초보자 경험 등)
- 변경 목적 또는 현재 의심되는 문제

## Output

- 위험도가 높은 축과 그 근거
- 우선 확인해야 할 테스트·리뷰·완화 조치
- 위험이 낮거나 근거가 부족한 축의 구분

## Zero residue

잔존 0 = **주요 리스크 신호 없음**. 현재 wiki 기준으로 충돌 가능성이 높은 축이 발견되지
않았다는 신호이며, 새 검증 항목은 선택 사항이다.

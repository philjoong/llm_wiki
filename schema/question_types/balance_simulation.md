# 밸런스 시뮬레이션

특정 빌드·세팅에서 클래스/스킬/아이템 조합이 어떻게 성능을 발휘하는지를 가상으로 검증한다.
실제 빌드 플레이 없이 밸런스 시트와 공식 페이지에서 추론한다.

## Input

- 빌드/세팅 구성 (클래스·스킬·장비·소비템 조합)
- 시나리오 (어떤 콘텐츠·PvP 모드에서)
- 비교 대상 (다른 빌드 또는 기준치)

## Output

- 시뮬레이션 결과 한 줄 + 인용된 공식·수치 페이지
- 기준 대비 강·약점 요약

## Zero residue

잔존 0 = **시뮬레이션에 필요한 수치 정보 부족**. 공식·계수·기준치 중 일부가 wiki에 정의되어 있지 않다는 신호.
"결과 없음"이 아니라 "사람이 데이터를 채워야 한다"는 출력이다.

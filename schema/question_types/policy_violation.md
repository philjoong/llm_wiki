# 정책 위반 탐지

사용자/운영 정책에 위반되는 동작·콘텐츠·상호작용을 식별한다.
정책 문서가 primary 근거이며, 시스템 룰은 보조 인용으로 다룬다.

## Input

- 의심 동작 (어떤 행위·콘텐츠가)
- 맥락 (어떤 상황에서 발생했는가)
- 적용 후보 정책 (어떤 정책이 관련 있는가)

## Output

- 위반된 정책 항목 인용
- 위반 정도 및 후속 조치 권고

## Zero residue

잔존 0 = **위반 없음 (긍정적 신호)**. 명시된 정책 어디에도 어긋나지 않았다는 결론.
"검색 실패"가 아니라 "정상"의 의미다.

# 게임 조작

현재 게임 상황과 목표에 맞춰 다음 조작이나 선택지를 제안한다. 단순 공략 검색이 아니라 화면 이해,
규칙, 쿨다운, 위험 지역, 타겟 상태를 종합해 지금 취할 행동을 고른다.

## Input

- 현재 상황 요약 (위치 / 대상 / 위험 요소 / 남은 자원)
- 플레이어 목표 (생존, 처치, 회피, 퀘스트 진행, 최적 딜 사이클 등)
- 사용 가능한 조작·스킬·아이템과 제약 조건

## Output

- 추천하는 다음 조작
- 대안 선택지와 선택 기준
- 관련 규칙·공략·시스템 근거 및 실패 위험

## Zero residue

잔존 0 = **추천 조작 근거 부족**. 현재 상황이나 사용 가능한 행동에 대한 wiki 근거가 부족해
다음 조작을 안정적으로 제안할 수 없다는 신호.

# 게임 화면 이해

현재 게임 화면이나 상황 설명을 바탕으로 어떤 상태가 벌어지고 있는지 파악한다. 캐릭터, 적,
UI 상태, 위치, 쿨다운, 경고 표시, 목표 정보를 조합해 관련 규칙과 현재 상황을 연결한다.

## Input

- 화면 설명 또는 스크린샷에서 읽힌 요소
- 현재 목표·퀘스트·전투 맥락
- HUD, 버프/디버프, 위치, 대상, 쿨다운 등 관찰 가능한 상태

## Output

- 현재 게임 상황 요약
- 관련 규칙·시스템 페이지 인용
- 확실한 판단과 추가 확인이 필요한 불확실성 구분

## Zero residue

잔존 0 = **상황 판단 근거 부족**. 화면 요소나 관련 규칙이 wiki에 충분히 연결되지 않아
현재 상황을 안정적으로 해석할 수 없다는 신호.

# 의견 전달

변경안이 의도에는 맞더라도 사용성, 학습 난이도, 오인 사용 가능성, 운영 부담 같은 정성적 리스크를
검토해 사람에게 전달할 의견을 만든다. 정책 위반이나 회귀처럼 명백한 실패가 아니어도 "의도에는
맞지만 초보자 영역에서 오인 사용 확률이 증가합니다" 같은 판단을 다룬다.

## Input

- 검토할 변경안 또는 현재 설계
- 의도한 효과와 대상 사용자
- 우려되는 플레이 맥락 또는 운영 맥락

## Output

- 의도와 맞는 부분
- 사용성·인지부하·오해 가능성에 대한 의견
- 사람이 결정해야 할 트레이드오프와 권고 문장

## Zero residue

잔존 0 = **전달할 추가 의견 없음**. 현재 wiki 기준으로 의도와 충돌하거나 별도 코멘트가 필요한
정성적 리스크가 발견되지 않았다는 신호.
